Gemini CLI

Futuristic Terminal Intelligence

Gemini CLI 是 Google 開源的終端機 AI 助理，讓你以最短路徑連接 Gemini 2.5 Pro。使用者可在終端內串接搜尋、檔案操作、Shell 指令與網頁抓取，並透過 MCP 擴充自訂工具。

立即體驗 NPM 套件資訊 
$ npx https://github.com/google-gemini/gemini-cli
$ gemini
> 幫我摘要昨天合併的 PR 變更

核心亮點

開箱即用的開發力

  • 免費額度：Google 帳號享 60 requests/min、每日 1,000 requests。
  • 一次擁有 100 萬 Token 的 Gemini 2.5 Pro 模型上下文。
  • 專為終端機設計的工作流程，支援 Shell 指令與檔案操作。

可擴充的 AI 代理人

  • 內建 Google Search、Web Fetch 等工具，獲得即時資訊。
  • 透過 Model Context Protocol 串接自訂服務，打造專屬工具鏈。
  • 提供會話檢查點與 GEMINI.md，自訂專案專屬語境。

快速安裝

立即啟動

# 無需安裝，直接使用
npx https://github.com/google-gemini/gemini-cli

# npm 全域安裝
npm install -g @google/gemini-cli

# Homebrew (macOS / Linux)
brew install gemini-cli

系統需求

  • Node.js 20 以上版本。
  • 適用 macOS、Linux、Windows 作業系統。
  • 支援 Preview / Stable / Nightly 三種發佈頻率。

發佈節奏

Preview

每週二 23:59 (UTC) 推出週更 Preview，快速體驗最新功能。

Stable

每週二 20:00 (UTC) 推出正式版，整合上一週 Preview 與修正。

Nightly

每日 00:00 (UTC) 推出 Nightly，直接跟上 main 分支的最新變更。

進階能力

程式理解與生成

  • 理解與重構大型程式碼庫。
  • 多模態生成：從 PDF、影像或草圖打造應用。
  • 自然語言除錯與說明。

自動化與整合

  • 在腳本或 CI 中無頭模式運行，提升自動化效率。
  • 透過 MCP 連結 GitHub、Slack、資料庫等服務。
  • GitHub Action 整合：審查 PR、議題分類、隨呼隨到。
